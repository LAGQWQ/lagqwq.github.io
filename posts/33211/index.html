Hexo绑定Github仓库方法

之前介绍 Hexo+Vercel 建站时，没有详细说明如何把 Hexo 站点和 Github 仓库绑定一起的方法。这次详细说一下。

注意：本文章是 Hexo+Vercel保姆级建站方法 的扩展，建议先阅读建站方法。

第一步-安装Git

前往 Git官网 下载并安装 Git 最新版。
